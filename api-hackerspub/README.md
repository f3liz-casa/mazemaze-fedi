# @f3liz/mazemaze-api-hackerspub

Typed [hackers.pub](https://hackers.pub) GraphQL API client.
Part of the [mazemaze](https://github.com/f3liz-casa/mazemaze-fedi) family —
one Melange runtime, with ReScript, TypeScript (JSDoc) and Melange surfaces.
Generated by `@f3liz/mazemaze-generator`.

## Install

```
npm install @f3liz/mazemaze-api-hackerspub
```

## TypeScript / JavaScript

```ts
import {
  connect,
  publicTimeline,
  personalTimeline,
  createNote,
  completeLoginChallenge,
} from "@f3liz/mazemaze-api-hackerspub";

// Unauthenticated client
const anon = connect("https://hackers.pub");
const feed = await publicTimeline(anon, { first: 20 });

// Authenticated client (Session.id from completeLoginChallenge)
const c = connect("https://hackers.pub", { token: "your-session-id" });
const home = await personalTimeline(c, { first: 20 });
const post = await createNote(c, {
  content: "Hello fediverse!",
  language: "en",
  visibility: "PUBLIC",
});
```

## Auth flow

1. Call `loginByUsername` or `loginByEmail` with a URI Template `verifyUrl`
   containing `{token}` and `{code}` placeholders.
2. The user receives a magic-link email; the link embeds the token and code.
3. Call `completeLoginChallenge({ token, code })` — on success,
   `result.data.completeLoginChallenge.id` is the bearer token.
4. Pass it as `opts.token` to `connect`.
